This archive contains the files with the electronic text of the Buxu Gaoseng Zhuan 補續高僧傳, by Minghe 明河, dated 1640. This edition was produced at the Library and Information Center of Dharma Drum Buddhist College and funded by the Haoran Foundation. It is based on the CBETA electronic text (CD 2010). We have added TEI/XML markup.
-------------------------------
Structure: The archive contains a wrapper file "wrapper-buxu.xml". The other files, one for each biography, are referenced from here via XInclude.
-------------------------------
Metadata: Please refer to the <teiHeader> in the wrapper file.
-------------------------------
Validation: Please use "gisSchema.rnc" to validate the wrapper file.
-------------------------------
